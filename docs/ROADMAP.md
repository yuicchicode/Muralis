# Roadmap

This roadmap is a direction, not a strict promise.

## Short term

- polish mobile layout;
- improve contextual menus;
- reduce visual noise in cards;
- review empty states;
- improve documentation;
- review import/export;
- improve basic accessibility.

## Mid term

- improve performance with many blocks;
- add stronger global search;
- improve command palette;
- add configurable shortcuts;
- improve visual export;
- create board templates;
- add board duplication;
- improve onboarding.

## Long term

Future ideas, not yet decided:

- presentation mode;
- export to image/PDF;
- optional sync;
- experimental local collaboration;
- simple plugin system;
- more complete PWA/offline support;
- advanced undo/redo history.

## Out of immediate scope

- required backend;
- authentication;
- framework migration;
- large dependencies;
- complex real-time collaboration.
