# Features

## Boards

The app supports multiple boards.

Each board has:

- name;
- active mode;
- viewport;
- blocks;
- associated Kanbans;
- associated Mind Maps.

## Blocks

Blocks are the central unit of Muralis.

Types:

```txt
idea
task
reference
goal
quote
link
```

## Free Board

The Free Board is the base mode.

It allows users to:

- create blocks;
- move blocks;
- resize blocks;
- edit blocks;
- select blocks;
- filter;
- search;
- use zoom and pan.

## Kanban

Kanban organizes tasks.

Features:

- multiple Kanbans per board;
- customizable stages;
- tasks by stage;
- drag and drop;
- filters;
- sorting;
- quick creation;
- add existing task;
- open on board.

## Matrix

Matrix organizes blocks by impact and effort.

Features:

- quadrants;
- create inside quadrant;
- drag between quadrants;
- filters;
- priority visualization.

## Mind Map

Mind Map connects blocks in a visual structure.

Features:

- multiple maps per board;
- nodes connected to blocks;
- child nodes;
- sibling nodes;
- edges;
- collapse/expand;
- existing block library;
- remove node;
- remove branch;
- open on board.

## Timeline

Timeline shows events derived from blocks.

Features:

- deadlines;
- creation dates;
- grouping by date;
- filters;
- sorting;
- quick task creation;
- quick milestone/goal creation;
- quick editing;
- open on board.

## Import/export

The app exports and imports JSON.

This allows:

- backup;
- manual migration;
- sharing;
- testing.

## Command palette

The command palette centralizes quick actions.

## Dark mode

The app supports light, dark, and system themes.
