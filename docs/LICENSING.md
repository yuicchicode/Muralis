# Licensing

Before publishing Muralis as open source, choose a license.

## Why does it matter?

Without an explicit license, other people may not have clear legal permission to use, modify, or distribute the project.

## Common suggestion

For simple open source projects, a common choice is:

```txt
MIT License
```

It is permissive and easy to understand.

## Other options

- Apache-2.0;
- GPL-3.0;
- AGPL-3.0;
- BSD-3-Clause.

## Recommendation

If the goal is to allow use, study, modification, and distribution with few restrictions, MIT is usually enough.

## Next step

When you decide, add a file named:

```txt
LICENSE
```

to the project root.

Also update the license section in `README.md`.
