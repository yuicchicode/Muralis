# Code of Conduct

Muralis aims to be an open, respectful, and collaborative project.

## Commitment

Participants are expected to contribute with respect, patience, and good faith.

People with different experience levels are welcome. The project also exists as a learning space.

## Expected behavior

- treat people with respect;
- explain technical decisions clearly;
- accept feedback;
- disagree without personal attacks;
- help beginners when possible;
- keep focus on the project and the user experience.

## Unacceptable behavior

- personal attacks;
- harassment;
- discriminatory language;
- public humiliation;
- spam;
- abusive behavior;
- aggressive insistence after a maintainer decision.

## Resolution

If an issue occurs, open an issue or contact the project maintainers.

The goal is to preserve a healthy environment for collaboration.
